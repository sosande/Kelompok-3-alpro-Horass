# Manajemen Administrasi Kelurahan Nagri Kaler

Program **Manajemen Administrasi Kelurahan Nagri Kaler** merupakan aplikasi berbasis *Command Line Interface (CLI)* yang dikembangkan menggunakan bahasa pemrograman **C**. Program ini bertujuan untuk membantu pengelolaan administrasi kelurahan secara terstruktur, meliputi data kependudukan, layanan surat, mutasi penduduk, retribusi, serta pembuatan laporan administrasi.

Proyek ini dikembangkan sebagai bagian dari tugas mata kuliah **Algoritma dan Pemrograman** pada Program Studi **Mekatronika dan Kecerdasan Buatan**, Universitas Pendidikan Indonesia Kampus Daerah di Purwakarta.
